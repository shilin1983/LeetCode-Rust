# leetcode-rust

- LeetCode 题解项目，使用 [Rust](https://www.rust-lang.org/) 语言实现。

- 帮助开发者更好地了解和学习数据结构与算法。

## 获取代码

```git
git clone https://github.com/shilin1983/leetcode-rust.git
```

## 安装依赖

```bash
cargo install cargo-llvm-cov
```

```bash
rustup component add llvm-tools-preview
```

## 运行测试

```shell
sh test.sh
```
